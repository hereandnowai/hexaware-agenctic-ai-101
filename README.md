## 🚀 Overview

This repository contains the chronological collection of lesson artifacts, hands-on labs, and assessment materials for the Agentic AI path.

- **Learn more at**: [hereandnowai.com](https://hereandnowai.com)

- **Days 6–7**: Quantization & LoRA Fine-tuning
